# 🌿 AI Plant Scanner App

An intelligent Flutter application that identifies plants using AI and provides detailed plant care information.

## ✨ Features

* 📷 Scan plants using camera
* 🌱 AI-powered plant identification
* 📝 Generate plant descriptions and care tips
* 📊 Display plant confidence score
* 🧠 Gemini AI integration
* 🏗 Clean Architecture
* 🔄 BLoC / Cubit State Management
* 💉 Dependency Injection using Injectable
